Структура проекта:
1. Controller  (классы с методами отвечающими за функционал проекта)
2. Framework(то что не относиться к конкретному проекту)
3. Model(база данных)
4. View(визуальное  представление)
5. Webroot(конфигурационные файлы)
Первая форма для добавления фильма
Вторая форма для сортировки
Третья форма для поиска по названию фильма или имени актера
Таблица с фильмами и их описанием, в ней можно редактировать названия, удалять строки. Кнопка Save сохраняет изменения . Кнопка Delete удаляет строку
